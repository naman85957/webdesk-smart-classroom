# Webdesk - Smart Classroom System

A full-stack web application for managing classroom activities like study materials, assignments, live classes, and student-teacher interaction.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## Features
- User authentication (Student & Teacher)
- Upload and access study materials
- Assignment creation and submission
- Live classes and scheduling
- Doubt discussion system

## Run Locally
```bash
npm run install-all
npm run dev
